Selected Research

Current and recent projects

Subgroup analysis for interval-censored data with a cure fraction

Developed a mixture promotion time cure model to capture heterogeneous covariate effects on cure probabilities and survival outcomes. Proposed a BIC-based subgroup analysis framework with efficient variance estimation using profiled likelihood and the delta method.

Survival Analysis · Cure Models · Interval Censoring

Variable selection for clustered mixed panel count data

Developed a unified modeling framework for recurrent event counts and binary outcomes while accounting for within-cluster correlation. Integrated regression and variable selection using the minimum information criterion and an L0 penalty.

Panel Count Data · Variable Selection · Clustered Data

Multi-state modeling for clinical trial observation intervals

At Novartis, implemented a time-inhomogeneous interval-censored multi-state model with piecewise-constant baseline hazards. Simulation and Inclisiran trial analyses showed improved estimation accuracy and informed observation-interval decisions in trial design.

Clinical Trials · Multi-State Models · Biostatistics

Bayesian hierarchical modeling for longitudinal PBC data

Applied Bayesian hierarchical longitudinal models in R and JAGS to Mayo Clinic PBC data, comparing fixed-effect, random-intercept, and random-slope models with frequentist alternatives and estimating patient-specific bilirubin trajectories via MCMC.

Bayesian Modeling · JAGS · Longitudinal Data
