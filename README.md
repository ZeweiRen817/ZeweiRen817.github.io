# Zewei Ren Personal Website

This is a static personal website prepared for GitHub Pages.

## Preview locally

Open `index.html` in a browser:

`C:\Users\zewei\personal-website\index.html`

The downloadable resume is stored at:

`assets\Zewei_Ren_Resume.pdf`

## Publish with GitHub Pages

1. Create a GitHub account if you do not already have one.
2. Create a new public repository named `<your-github-username>.github.io`.
3. Upload `index.html` and `styles.css` to that repository.
4. Open `https://<your-github-username>.github.io` after GitHub Pages finishes publishing.

Optional later:

- Add a GitHub profile link if you want it public.
- Add Google Scholar if you have a public profile.
- Add links to publications or preprints.
- Buy a custom domain such as `zeweiren.com` and connect it to GitHub Pages.
